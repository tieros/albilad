# Audit Domain: Foundations / Typography

Use this domain for type systems, font logic, text styles, readability, and multilingual readiness.

## Inspect

- typography scale
- naming consistency
- line-height discipline
- role clarity between display, title, body, label, and assistive text
- Arabic and Latin support
- font fallback behavior
- tokenization and documentation

## Common Failure Modes

- unclear type hierarchy
- missing semantic role definitions
- weak multilingual support
- inconsistent line heights
- implementation mismatch between design and runtime
