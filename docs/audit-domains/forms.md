# Audit Domain: Components / Forms

Use this domain for input and form-entry components.

## Typical Scope

- text inputs
- select inputs
- labels
- text areas
- amount inputs
- telephone inputs
- upload file
- auth inputs
- calendar and date fields

## Inspect

- anatomy consistency
- validation states
- helper and error messaging
- formatting logic
- prefix and suffix handling
- mode and locale readiness
