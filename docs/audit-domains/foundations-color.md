# Audit Domain: Foundations / Color

Use this domain for token, palette, semantic color, theme, and contrast issues.

## Inspect

- semantic color separation
- brand versus system color conflicts
- light and dark token logic
- state colors
- contrast compliance
- special occasion theming
- color documentation quality

## Common Failure Modes

- brand colors used as system feedback colors
- missing semantic tokens
- contrast failures
- raw color usage instead of tokens
- incomplete theme coverage
