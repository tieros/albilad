# Audit Foundations Skill

Use this skill when the issue belongs to design-system foundations rather than component families.

## Required Reads

- `docs/audit.md`
- `docs/audit-domains/foundations-color.md`
- `docs/audit-domains/foundations-typography.md`
- `docs/audit-domains/foundations-spacing.md`
- `docs/audit-domains/foundations-border.md`
- `docs/audit-domains/foundations-elevation.md`

## What This Skill Must Do

1. Route the issue to the correct foundation subdomain.
2. Check whether the finding already exists in the Foundations section of the audit.
3. Reuse or extend before creating a new finding.
4. Preserve the existing audit structure.
