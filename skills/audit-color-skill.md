# Audit Color Skill

Use this skill when the task is specifically about color, semantic color, theming, or contrast in the Albilad design system.

## Required Reads

- `docs/audit.md`
- `docs/audit-domains/foundations-color.md`

## What This Skill Must Check

- contrast and accessibility
- semantic versus brand color separation
- theme readiness
- token completeness
- state colors
- documentation quality

## Continuation Rule

Before writing anything new, inspect whether the issue is already represented by:

- contrast accessibility
- brand versus system color separation
- special occasion palettes
- documentation issues

If yes, extend instead of duplicating.
