# Audit Typography Skill

Use this skill when the task is specifically about type systems, text styles, fonts, or multilingual typography behavior.

## Required Reads

- `docs/audit.md`
- `docs/audit-domains/foundations-typography.md`

## What This Skill Must Check

- scale clarity
- text style hierarchy
- line-height consistency
- Arabic and Latin compatibility
- fallback behavior
- token and documentation quality
