# Audit Components Skill

Use this skill when the issue belongs to a component category rather than foundations, assets, or patterns.

## Required Reads

- `docs/audit.md`
- the relevant component domain file in `docs/audit-domains/`

## What This Skill Must Do

1. Route to the correct component family.
2. Review the delivered audit findings for that family.
3. Review the original Albilad DS implementation.
4. Reuse, extend, or create findings without breaking consistency.
